UC9: Расчет зарплаты

Акторы:

Система (внутренний процесс Payroll) – обрабатывает данные и производит расчет.

(Непрямо участвуют Уборщик и Telegram Bot через передачу данных.)

Предусловия:

Все данные по выполненным уборкам, включая подтвержденные записи и отзывы, накоплены в базе данных за отчетный период.

Триггер: Завершение отчетного периода (например, конец месяца) или запрос со стороны финансового отдела.

Основной поток событий:

Payroll-система инициирует агрегирование данных за указанный период: количество и площадь выполненных уборок, оценки качества (на основе отзывов).

На основании установленных тарифов и бизнес-правил производится расчет заработной платы для каждого уборщика.

Результаты расчета фиксируются и передаются в платежную систему для проведения выплаты.

Уведомление о произведенной выплате отправляется соответствующему сотруднику.

Постусловия:

Зарплата для каждого сотрудника рассчитана и задокументирована, данные переданы для осуществления выплаты, что завершает полный цикл процесса.
