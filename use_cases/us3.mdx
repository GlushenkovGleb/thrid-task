UC3: Сформировать отчет

Актор:

Менеджер – получает агрегированную информацию по выполненным заданиям для контроля работы.

Предусловия:

Менеджер имеет доступ к административному интерфейсу системы.

В системе накоплены данные по уборкам за выбранный период.

Триггер: Требование к формированию отчета по итогам работы за определенный период.

Основной поток событий:

Менеджер инициирует процесс формирования отчета через веб-интерфейс.

Он задает критерии отчета (период, объекты, сотрудники, показатели качества).

Система собирает и агрегирует данные из базы.

Отчет генерируется в удобном для анализа виде (таблицы, графики) и представляется менеджеру.

По необходимости отчет может быть экспортирован (например, в формат PDF или Excel).

Постусловия:

Менеджер получает готовый отчет, который можно использовать для анализа эффективности работы и принятия управленческих решений.
