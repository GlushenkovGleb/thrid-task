UC8: Фиксация уборки

Актор:

Telegram Bot – центральный компонент для приема данных от мобильного приложения.

Предусловия:

Данные от мобильного устройства (например, после сканирования QR-кода) получены и готовы к обработке.

Триггер: После сканирования QR-кода мобильным устройством, данные отправляются в систему.

Основной поток событий:

Telegram Bot принимает данные сканирования, включающие идентификатор уборщика, помещения и время уборки.

Бот сверяет полученную информацию с назначенными заданиями.

При успехе бот отправляет данные для записи в базу данных и подтверждает фиксацию уборки.

Альтернативные потоки:

Если данные не соответствуют назначению, бот отправляет уведомление об ошибке и информирует менеджера для выяснения причин.

Постусловия:

Факт уборки зарегистрирован и доступен для последующей обработки (например, для расчета зарплаты).
